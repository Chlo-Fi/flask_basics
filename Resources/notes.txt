Installing Virtual Environment on Windows:
[[Brackets are meant to show variables that you would fill in on your own, ignore them in the actual code]]

1. Command Prompt
2. pip install virtualenv
3. mkdir [[directory name]]
4. cd [[directory name]]
5. virtualenv [[virtual environment name - i.e. flaskenv]]
6. Activate it by typing in [[virtual environment name]]\scripts\activate
